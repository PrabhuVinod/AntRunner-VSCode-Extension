# Change Log

All notable changes to the "ant-target-runner" extension will be documented in this file.

Check [Keep a Changelog](http://keepachangelog.com/) for recommendations on how to structure this file.

## [0.1.0] - 2026-03-04

- Read build.xml and display runnable Ant targets in the Activity Bar
- Run Ant targets directly from the sidebar
- Open target definition in editor
- Toggle between primary and all targets
- Search/filter targets
- Favourite targets for quick access
- Configure custom build file path
- Verbose mode support
